◆ FixedUpdate()

void UnitController.FixedUpdate ( )

If a unit has moved, it will be added to the damageableAddQueue. This method adds the unit to the correct index in damageableLocations. At locations where units were added or deleted, this method will iterate through those locations and their neighbors and update the attack targets for any of the units in these positions
